Lista de afazeres em React

O aplicativo deve ser protegido por uma senha.

Deve ser criado uma interface para listar, criar, atualizar e remover tarefas. Cada tarefa é composta de um identificador, um nome e uma descrição.

Condições de desenvolvimento

Deve ser usado React, Redux e React Router.

O design e os componentes podem ser feitos à mão, ou usando bibliotecas de componentes como Material UI ou shadcn/ui
